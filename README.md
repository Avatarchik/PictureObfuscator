Obf A Pic Or DeObf A Pic.

### 使用方法：
#### 1.下载Release运行 或 克隆源码进行编译运行。
#### 2.输入input文件名（必须是绝对路径）。
#### 3.输入是混淆还是反混淆（两者是相反的）。
#### 4.输入Seed(密码)。
#### 5.等待程序运行结果。
